# finflow-app

## Environment

This app is configured to use Supabase and can run with the built-in defaults in `src/integrations/supabase/client.ts`, or with explicit Vite env vars:

- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`
